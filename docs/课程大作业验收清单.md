## 建议答辩 Demo 流程

1. 启动 `pnpm dev`。
2. 浏览器打开 `http://localhost:5173`。
3. 输入 `分析 ORDI`。
4. 点击 ORDI 候选，展示执行卡和“AI 助手”。
5. 提问“分析 ORDI，下一步应该看什么？”。
6. 展示证据页、回放时间线、人工复核落盘。
7. 运行 `python -m pytest tests/requirements_test.py` 展示 TDD-for-AI。
